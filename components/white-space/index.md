WhiteSpace 是 Flex 布局的控件之一。

## API

| 成员        | 说明           | 类型      |  可选值         | 默认值       |
|------------|----------------|--------------------|--------------|
| size       |  上下留白的间距   | string | `xs`、`sm`、`md`、`lg`、`xl` | `md`  |
